# Internet Safety Course

Статический сайт для GitHub Pages:

- тест на 10 вопросов;
- подсчет баллов и показ одного из 4 результатов;
- вводный экран и 5 последовательных уроков;
- финальный экран с итогами курса.

## Локальный запуск

Откройте файл `index.html` в браузере.

## Публикация

Проект рассчитан на публикацию через GitHub Pages из ветки `main`.
